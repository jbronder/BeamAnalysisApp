# Beam Analysis App
A C# re-implementation of a React application I wrote 
[Beam Analysis Tool](https://jbronder.github.io/beam-analysis-tool/),
[GitHub repo](https://github.com/jbronder/beam-analysis-tool)

## Tools & Technologies
- [AvaloniaUI](https://avaloniaui.net/)
- [LiveCharts2](https://livecharts.dev/)

## About
A structural beam calculator written in a cross-platform WPF-compatible
framework. Dyanamially computes the maximum/minimum values for the load cases
and generates shear, moment, and deflection diagrams.
